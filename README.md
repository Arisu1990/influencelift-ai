## 📊 Using the dashboard

The dashboard organizes your work into three main areas.

### Cleaning data
Influencer data often arrives in messy formats. Use the Data Import tab to upload your spreadsheets. The system detects errors and fixes them. It removes duplicate entries and checks for missing information. This ensures your analysis remains accurate.

### Forecasting sales
You can predict how much product you will sell after a campaign. Go to the Sales Forecast tab and select your campaign dates. The application builds a model based on your past sales history. It displays a graph showing the expected increase in orders.

### Simulating decisions
Test your ideas before you spend money. Use the Simulator tab to adjust your budget and influencer choices. The application shows the likely outcome of each change. You can compare different plans to see which one brings the best return.

## 🛠 Solving common issues

Most problems resolve with simple steps.

*   The program does not open: Restart your computer and try again.
*   Data upload fails: Ensure your file is in the .csv or .xlsx format.
*   Graphs appear blank: Check that your spreadsheet includes date and sales columns.

If you encounter errors, check the logs folder inside the installation directory. You can share these files if you ask for help.
